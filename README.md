# Развертывание и запуск веб-приложения

1. Установить GIT, если еще не установлен (временно весь софт размещен в папке `\\titanium\Incoming\Software`)
2. В локальную папку (далее будем называть это рабочий каталог) выгрузить содержимое текущего GIT репозитария
3. Установить NodeJS
4. Выполнить команду в корне рабочего каталога
### `npm install`
Команда выполняется долго, но можно предварительно скопировать из софта архив node_modules.zip, распаковать и разместить содержимое в корне рабочего каталога. Так же следует перед выполнением команды удалить файл package-lock.json
5. Запуск сервера разработки выполняется командой
### `npm run start`
или запуском
### `run.cmd`

Может потребоваться добавить каталог `./node_modules/react-script/bin` в переменную окружения PATH
Так же в корне необходимо создать файл .env и прописать в него адрес сервера
REACT_APP_DEV_ENDPOINT=http://localhost:8081
REACT_APP_PROD_ENDPOINT=http://core.srv.gelicon.biz:8081

6. Результат доступен по URL
### `localhost:3000`
